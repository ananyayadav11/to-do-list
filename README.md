# to-do-list
# 📝 To-Do List Web App

A simple and minimal To-Do List application built using HTML, CSS, and JavaScript. Users can add, mark, and delete tasks easily.

#  Technologies Used

- HTML – Structure  
- CSS – Styling  
- JavaScript – Functionality / Interactions
